GRID-COLUMN

Define en qué columna empieza y termina un elemento dentro del grid

GRID-ROW

Define en qué fila empieza y termina un elemento.

GRID-TEMPLATE-COLUMNS

Establece el número, tamaño y comportamiento de las columnas del grid.

GRID-TEMPLATE-ROW

Establece el número, tamaño y comportamiento de las filas.

GRID-AEREA

Define un área del grid que puede incluir múltiples filas y columnas

JUSTIFY-SELF

Alinea un elemento dentro de su celda horizontalmente.

ALIGN-SELF

Alinea un elemento dentro de su celda verticalmente.

JUSTIFY-ITEMS

Alinea todos los elementos dentro de sus celdas horizontalmente.

ALIGN-ITEMS

Alinea todos los elementos dentro de sus celdas verticalmente.

CSS GRID


CSS Grid Layout o CSS Grid es un sistema de diseño bidimensional que permite distribuir elementos HTML en filas y columnas con gran precisión y flexibilidad.


¿Para qué sirve CSS Grid?


Su principal función es facilitar la creación de layouts responsivos, es decir, que se adapten automáticamente a distintos tamaños de pantalla (como computadoras, tabletas o celulares), usando unidades como fr,auto y funciones como minmax().


Características de CSS Grid


• Diseño bidimensional: A diferencia de Flexbox, que solo maneja una dirección a la vez, Grid permite controlar tanto el eje horizontal como el vertical.


• Distribución precisa: Puedes ubicar cada elemento exactamente donde lo necesitas usando líneas del grid, sin depender del orden del HTML.


• Separación clara de contenido y presentación: El HTML contiene solo el contenido, y el CSS se encarga completamente del diseño y la colocación de los elementos.


• Facilidad para el diseño responsivo: Permite adaptar el diseño a diferentes pantallas sin necesidad de reestructurar el HTML.
