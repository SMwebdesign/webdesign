CURSO DISEÑO WEB

HTML 5

HTML5 is the latest version of Hypertext Markup Language, used for structuring and presenting content on the web. It introduces new elements, attributes, and behaviors, providing more powerful capabilities for web developers. HTML5 enhances multimedia support, improves accessibility, and allows for better interaction without requiring additional plugins. Notable features include the canvas element for graphics, the video and audio elements for media embedding, and improved support for mobile-friendly designs.

  • Semantic Elements:New tags like article, section, and header improve readability and accessibility.
  • Web Workers:Enables background scripts to run without affecting performance.
  • Form Enhancements:New input types like email, date, and number improve user experience.
  • WebSockets:Enables real-time communication between client and server.
  • Multimedia Support:Native audio and video elements allow embedding media without third-party plugins.

CSS

CSS (Cascading Style Sheets) is a language used to style and format web pages. It works alongside HTML by defining how elements should look, including colors, fonts, spacing, and layout. With CSS, developers can create visually appealing and responsive designs, ensuring websites look great on different devices and screen sizes.

  • Selectors & Properties:CSS allows precise styling of HTML elements using selectors and properties.
  • Flexbox & Grid:Provides powerful layout systems for responsive and dynamic designs.
  • Animations & Transitions:Enables smooth visual effects without JavaScript.
  • Media Queries:Allows adaptive styling based on screen size and device type.
  • Accessibility Features:Improves usability for users with disabilities.
